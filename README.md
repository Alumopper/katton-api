# Katton

This is the quick start guide for [Katton](https://github.com/Alumopper/Katton). Katton is a powerful modding framework for Minecraft that allows you to create mods using Kotlin scripts. It provides a wide range of APIs and tools to help you develop your mods quickly and efficiently. Whether you're a beginner or an experienced modder, Katton has something to offer for everyone.
