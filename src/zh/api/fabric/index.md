# Fabric API

从模块`fabric`生成。

## 源码根目录

- `fabric/src/main/kotlin/top/katton/api`

## 页面

- [ChunkAndBlockEvent](./event/ChunkAndBlockEvent.md) - Fabric 平台的区块、方方块实体和方方块事件。
- [ItemComponentEvent](./event/ItemComponentEvent.md) - Fabric 平台的物品组件和附魔事件。
- [ItemEvent](./event/ItemEvent.md) - Fabric 平台的物品交互事件。
- [LivingBehaviorEvent](./event/LivingBehaviorEvent.md) - Fabric平台的生命体行为事件。
- [LivingUseItemEvent](./event/LivingUseItemEvent.md) - 生命实体使用Fabric平台的物品事件。
- [LootTableEvent](./event/LootTableEvent.md) - Fabric 平台的战利品表事件。
- [PlayerEvent](./event/PlayerEvent.md) - Fabric 平台的玩家交互事件。
- [ServerEntityCombatEvent](./event/ServerEntityCombatEvent.md) - Fabric平台的服务器实体战斗事件。
- [ServerEntityEvent](./event/ServerEntityEvent.md) - Fabric 平台的服务器实体生命周期事件。
- [ServerEvent](./event/ServerEvent.md) - Fabric 平台的服务器生命周期事件。
- [ServerLivingEntityEvent](./event/ServerLivingEntityEvent.md) - Fabric 平台的服务器端生命实体事件。
- [ServerMessageEvent](./event/ServerMessageEvent.md) - Fabric 平台的服务器端消息事件。
- [ServerMobEffectEvent](./event/ServerMobEffectEvent.md) - Fabric 平台的生物效果事件。
- [ServerPlayerEvent](./event/ServerPlayerEvent.md) - Fabric 平台的服务器玩家事件。
- [FabricManagedEvents](./event/managed/FabricManagedEvents.md) - Fabric 实施[ManagedListenerProvider]。
