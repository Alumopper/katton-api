# Paper API

从模块`paper`生成。

## 源码根目录

- `paper/src/main/kotlin/top/katton/api`

## 页面

- [ChunkAndBlockEvent](./event/ChunkAndBlockEvent.md) - Paper（Bukkit）平台的分块和阻止事件。
- [ItemComponentEvent](./event/ItemComponentEvent.md) - Paper (Bukkit) 平台的物品组件事件。
- [ItemEvent](./event/ItemEvent.md) - Paper (Bukkit) 平台的物品交互事件。
- [LivingBehaviorEvent](./event/LivingBehaviorEvent.md) - Paper（Bukkit）平台的生活行为事件。
- [LivingUseItemEvent](./event/LivingUseItemEvent.md) - Paper（Bukkit）平台的生命实体物品使用事件。
- [LootTableEvent](./event/LootTableEvent.md) - Paper (Bukkit) 平台的战利品表事件。
- [PlayerEvent](./event/PlayerEvent.md) - Paper（Bukkit）平台的玩家交互事件。
- [ServerEntityCombatEvent](./event/ServerEntityCombatEvent.md) - Paper（Bukkit）平台的服务器实体战斗事件。
- [ServerEntityEvent](./event/ServerEntityEvent.md) - Paper (Bukkit) 平台的服务器实体事件。
- [ServerEvent](./event/ServerEvent.md) - Paper (Bukkit) 平台的服务器生命周期事件。
- [ServerLivingEntityEvent](./event/ServerLivingEntityEvent.md) - Paper（Bukkit）平台的服务器活体事件。
- [ServerMessageEvent](./event/ServerMessageEvent.md) - Paper (Bukkit) 平台的服务器消息事件。
- [ServerMobEffectEvent](./event/ServerMobEffectEvent.md) - Paper（Bukkit）平台的服务器生物效果事件。
- [ServerPlayerEvent](./event/ServerPlayerEvent.md) - Paper（Bukkit）平台的服务器玩家事件。
- [PaperManagedEvents](./event/managed/PaperManagedEvents.md) - Paper（Bukkit）[ManagedListenerProvider]的实现。
